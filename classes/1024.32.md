This defines the response from a Level II node for a [CLASS1.PROTOCOL, Type=32, Who is there?](./class1.protocol.md#type_31_0x1f_who_is_there) event.

 | Byte  | Description                                       | 
 | :----:  | -----------                                       | 
 | 0-15  | GUID for node                                     | 
 | 16-47 | MDF of node                                       | 
 | 3     | 0x00 which is the least significant byte of 32000 |
