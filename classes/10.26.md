**Default unit:** gray (Gy).   
**Opt unit:** sievert (Sv) (1).

This is a measurement of a radiation dose (Absorbed dose of ionizing radiation). 

 | Data byte | Description                         | 
 | :---------: | -----------                         | 
 | 0         | Data coding.                        | 
 | 1-7       | Data with format defined by byte 0. |
